# SnowCone
This is a SnowCone app which has the sole purpose of making snow cones xd.

Frontend: TypeScript, React, TailwindCSS.
Backend: ExpressJS.
Database: MongoDB.
API: REST with GraphQL.
Assests: Firebase/AWS.
Caching: Redis.
Authentication: Google/Microsoft and self.
Payment: Gateway integrations.

SETUP:
Web-
1. Install react project with create-react-app with typescript using:
npx create-react-app my-app --template typescript

2. Setup tailwindcss using:
https://tailwindcss.com/docs/guides/create-react-app
